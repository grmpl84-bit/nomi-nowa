# FocusRemind

**Voice-powered reminder app designed for ADHD minds.**

Quick capture, smart scheduling, zero friction.

## Features
- Voice Input - tap mic, speak, done
- Smart Time Parsing - "za 5 minut", "tomorrow at 3pm"
- Quick Intervals - 5, 10, 15, 30 min, 1h, 2h
- Nag Mode - repeats until completed
- Streak gamification
- Home screen widget
- Material Design 3 + dynamic colors
- Multi-language (EN/PL)

## Tech Stack
- Kotlin + Jetpack Compose + Material 3
- Room Database + Hilt DI
- WorkManager for scheduling
- Jetpack Glance widget
- Android SpeechRecognizer
- Min SDK 26 (Android 8.0)

## Building
1. Open in Android Studio (Hedgehog or newer)
2. Sync Gradle
3. Run on device/emulator (API 26+)
